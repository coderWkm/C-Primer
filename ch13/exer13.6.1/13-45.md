### 13.45
- 左值引用：与右值引用相反（如普通变量，绑定的对象**持久**）
- 右值引用：绑定到字面常量、返回右值的表达式、转换的表达式（这些对象**即将销毁**）
### 13.46
```cpp
int f();
vector<int> vi(100);
int &&r1 = f();								// f 是返回非引用类型的函数，返回值是一个右值
int &r2 = vi[0];							// 下标运算返回的是左值
int &r3 = r1;									// r1 是一个变量，而变量是左值
int &&r4 = vi[0] * f();				// vi[0] * f() 是一个算术运算表达式，返回右值
```
